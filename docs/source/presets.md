# Presets

The Presets layer provides high-level constructors that bundle geometry generation, energy setup, and contact registration into a single call.
They are the recommended starting point for most scenes.

Access via `simulation.presets->deformables` and `simulation.presets->rigidbodies` (C++) or `simulation.presets().deformables()` / `simulation.presets().rigidbodies()` (Python).

## Handler Types

All preset constructors return a **VCH** (Vertex-Connectivity-Handler) struct containing:

| Field | Contents |
|---|---|
| `vertices` | The generated 3D vertex positions |
| `triangles` / `segments` / (none for volumes) | The generated connectivity |
| `handler` | Sub-handlers for each registered model |

The `handler` field is the main object you keep for further interaction:

```cpp
auto vch = simulation.presets->deformables->add_surface_grid("cloth", size, {20,20}, params);
auto& h = vch.handler;     // Surface::Handler
auto& ps = h.point_set;    // PointSetHandler
auto& contact = h.contact; // ContactHandler
```

## Deformable Presets

### Surface Grid (Cloth)

Creates a flat, axis-aligned triangle mesh with uniform triangulation:

```cpp
// C++
auto vch = simulation.presets->deformables->add_surface_grid(
    "cloth",
    Eigen::Vector2d(0.5, 0.5),  // physical size in X and Y (m)
    {30, 30},                    // vertex count along each axis
    stark::Surface::Params::Cotton_Fabric()
);
```

```python
# Python
vV, vC, vH = simulation.presets().deformables().add_surface_grid(
    "cloth",
    np.array([0.5, 0.5]),
    np.array([30, 30]),
    pystark.Surface.Params.Cotton_Fabric()
)
```

For a custom mesh:

```cpp
auto h = simulation.presets->deformables->add_surface(
    "cloth", vertices, triangles, params
);
```

### Volume Grid (Tet Box)

Creates a tetrahedral grid filling an axis-aligned box:

```cpp
auto vch = simulation.presets->deformables->add_volume_grid(
    "body",
    Eigen::Vector3d(0.1, 0.1, 0.2),  // physical size (m)
    {3, 3, 6},                         // subdivisions along each axis
    stark::Volume::Params()
);
```

For a custom tet mesh (e.g. generated by TetGen or Gmsh):

```cpp
auto h = simulation.presets->deformables->add_volume("body", vertices, tets, params);
```

### Line Segments (Rod)

A straight rod made of uniform segments:

```cpp
auto vch = simulation.presets->deformables->add_line_as_segments(
    "rod",
    Eigen::Vector3d(0, 0, 0),   // start point
    Eigen::Vector3d(1, 0, 0),   // end point
    20,                          // number of segments
    stark::Line::Params::Elastic_Rubberband()
);
```

For arbitrary geometry:

```cpp
auto h = simulation.presets->deformables->add_line("rod", vertices, segments, params);
```

### Prescribed Surface

A surface whose positions are fully prescribed (zero DOF — moves exactly as scripted):

```cpp
auto h = simulation.presets->deformables->add_prescribed_surface(
    "obstacle", vertices, triangles,
    stark::PrescribedSurface::Params()
);
```

## Rigid Body Presets

All rigid body presets compute the inertia tensor for the chosen shape automatically.

### Box

```cpp
// Uniform box
auto vch = simulation.presets->rigidbodies->add_box("box", mass, 0.1);  // cube side = 0.1 m
// Non-uniform box
auto vch = simulation.presets->rigidbodies->add_box("box", mass, Eigen::Vector3d(0.1, 0.2, 0.3));
```

### Sphere

```cpp
auto vch = simulation.presets->rigidbodies->add_sphere("ball", mass, radius, subdivisions=2);
```

### Cylinder

```cpp
auto vch = simulation.presets->rigidbodies->add_cylinder(
    "cylinder", mass, radius, full_height, slices=16, stacks=1
);
```

### Torus

```cpp
auto vch = simulation.presets->rigidbodies->add_torus(
    "torus", mass, outer_radius, inner_radius, slices=16, stacks=32
);
```

### Custom Mesh

For arbitrary collision and render meshes:

```cpp
auto h = simulation.presets->rigidbodies->add(
    "body", mass, inertia_tensor_local,
    collision_vertices, collision_triangles,
    render_vertices, render_triangles,
    contact_params
);
```

Or with a shared mesh for both collision and render:

```cpp
auto h = simulation.presets->rigidbodies->add(
    "body", mass, inertia_tensor_local,
    vertices, triangles
);
```

## Built-in Material Presets

### Surface (Cloth)

| Preset | Description |
|---|---|
| `Surface::Params::Cotton_Fabric()` | A soft, moderately stiff cloth material |

### Line (Rod)

| Preset | Description |
|---|---|
| `Line::Params::Elastic_Rubberband()` | A highly elastic rubber-like rod |

Custom parameters can be constructed from scratch or by modifying a preset:

```cpp
auto params = stark::Surface::Params::Cotton_Fabric();
params.strain.youngs_modulus = 2e5;   // stiffer than default
params.inertia.damping       = 2.0;   // more damping
```
